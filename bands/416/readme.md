## Slot

每次slot给到一个长度为5的带子片段，实际牌面长度为3。从带子片段中取牌面的时候偏移到index=1的位置开始，取3个icon。

绿猪触发的free spin中有两种大小的牌面，共用一个带子。其中牌面长度为4的时候偏移量为1，牌面长度为5的时候无需偏移。
